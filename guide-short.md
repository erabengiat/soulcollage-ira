### מסך הבית — הכפתורים

| כפתור | מה הוא עושה |
|---|---|
| 🔍 **צפייה בקלפים** | מצגת · תמונות ממוזערות · לפי יומנים |
| 🎴 **שליפת קלף אחד** | שולף קלף אקראי אחד |
| 🃏 **קריאת ארבעה קלפים** | קריאה עם 4 עמדות + שאלה |
| 🕐 **קריאות קודמות** | הקריאות ששמרתם |
| ➕ **הוספת קלף** | מוסיף קלף חדש (מצלמה / גלריה / קבצים) |
| 🔎 **חיפוש קלף** | חיפוש לפי מילה, שם, או "נוסף לאחרונה" |
| ⚙️ **הגדרות** | יומן חדש · ביטול פעולה · לוג · זמן שימוש |
| 📔 **היומנים** | כל היומנים והקלפים שבהם |

### 5 הפעולות הנפוצות

**לראות קלף**
היומנים ← בחרו יומן ← לחצו על קלף

**להגדיל קלף למסך מלא**
**לחיצה כפולה** על התמונה. לחיצה נוספת סוגרת.

**לערוך קלף**
פתחו קלף ← לחצו על ✏️ שליד השם ← שנו ← "שמירת השינויים"

**קריאת 4 קלפים**
קריאת ארבעה קלפים ← כתבו שאלה (או הקישו על 🎤 והכתיבו בקול) ← "שליפת ארבעה קלפים" ← "שמירת הקריאה"

**להוסיף קלף**
הוספת קלף ← בחרו מקור (מצלמה / גלריה / קבצים) ← שם ← יומן ← סדרה ← הערה ← שמירה

### טיפים

- 🏠 **כפתור הבית** (עיגול קטן למטה) מחזיר תמיד למסך הבית

- **הסדרה** ניתנת לשינוי ישירות בתצוגת הקלף
- **יומן חדש** — בהגדרות, או בתוך "הוספת קלף"
